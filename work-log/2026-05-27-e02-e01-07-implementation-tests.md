## Code Changes

### 1) RBAC Query Contract (E02-02)
Implemented strict E01-07 list contract for role permission matrix.

- Updated `back-end/src/rbac/rbac.types.ts`:
  - added `RolePermissionsFilterInput`
  - added `RolePermissionsQueryInput` (`pagination`, `orderBy`, `filter`)
  - added paginated payload `RolePermissionsPayload` (`data`, `rolePermissions`, `totalCount`, `pageInfo`)

- Updated `back-end/src/rbac/rbac.resolver.ts`:
  - changed `rolePermissions` to accept `query?: RolePermissionsQueryInput`
  - added filter/order/pagination logic using shared E01-07 primitives

  - validates unsupported `orderBy.field` with `BadRequestException`

- `back-end/src/schema.gql` regenerated by code-first GraphQL bootstrap during test execution.
